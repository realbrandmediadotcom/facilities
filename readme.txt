== Description ==
The Facilities plugin is the best way to provide campground visitors with a full description of what the campground has to offer. Provide your visitors with a full description of each of your campground facilities, site amenities, upcoming events, announcements and any other useful information. The plugin allows for a full description, along with bullet points, photographs and helpful URL links. 


== Installation ==
Upload the Facilities plugin to your blog, activate it, and copy the shortcode - [facilities] to a new page.

Setting up the facilities plugin.




Main Sections:

1. Top Section tab.

Enter a main title for the campground facilities or amenities. e.g. Facilities at The Family Campground

Enter a good description. e.g. We offer first-class amenities, activities, and services. The Family Campground is a community designed to fulfill every wish of todayâ€™s discerning RV owner. Residents and visitors in this wonderful resort community will have access to a complete suite of luxury amenities.

Make sure to save changes. 


2. Bottom Section tab. 

1. Content Section One (1)

If you do not want to display the content from section one, make sure that the checkbox is unchecked. Otherwise the content from section one will be displayed on the site.

Enter the main title. e.g. Events Calendar

Enter a meaningful description. Click the button below to view our events calendar.

Add a descriptive name that will be shown on the button, this is the button text. e.g. Events Calendar

Add the URL link to the button after it has been clicked. e.g. https://example.com

If you do not want the URL link to open into a new tab, uncheck the checkbox next to Open Link in New Tab.

The delete fields button will delete all the fields data from the database table.

Make sure to save changes.

2. Content Section Two (2)

If you do not want to display the content from section two, make sure that the checkbox is unchecked. Otherwise the content from section two will be displayed on the site.

You may adjust the height of the element. You can choose from 250 - 550 pixels by scrolling the toggle.

Enter the main title. e.g. CONTACT US TODAY.

You can change the title text color.

Enter a meaningful description. If you have any questions, please feel free to reach out to us today. We are here to help answer any question that you might have.

You can change the description text color.

Add a descriptive name that will be shown on the button, this is the button text. e.g. Get in Touch

Add the URL link to the button after it has been clicked. e.g. https://example.com

If you do not want the URL link to open into a new tab, uncheck the checkbox next to Open Link in New Tab.

If you want to add a background image to the element, click on the upload image button and select an image from the media library or upload an image of your choice.

If you no longer want to use the selected image, click on the checkbox - Check to remove image and click Save Changes.

You can choice a color for the background color of the element.

The background overlay can be used with either background image or without an image selected.

You can change the background overlay opacity. You have a choice from 0 - 1 range.

The delete fields button will delete all the fields data from the database table.

Make sure to save changes.


3. Settings tab.

You have a choice to either show the data with animation or as a static layout.

If you want to show your visitors, the plugin data with animation make sure the checkbox is checked.

If the checkbox is unchecked the displaying of the plugin will show as a static layout.

Make sure to save changes.



Categories:

1. Add New Category

Create a new category. e.g. Facilities

Create a new title. e.g. Campground Facilities

Upload an Image or photograph that you would like to show your visitors.

Add the name that will appear on the button text. e.g. BOOK ONLINE

Add the button URL. e.g. https://example.com

Make sure to click the Add Category button.

If you want the new category to be seen on the frontend of the site, make sure that the checkbox is checked for enable category.

In the table under enabled, you will see either yes or no for the category.

**Enabled** to view a category on the frontend, you must have Enabled set to Yes.

To change enabled click, edit under the Category in the list table. The checkbox must be checked to be enabled. You will find this next to Enable this category to display it on the site.

Make sure to save changes.


2. Editing Category

To edit you will find the edit link under the category name in the table row. Click on edit to go to the edit category section.

To edit a category name, make the changes you want in the text field. The plugin will not allow you to have the same two category names.

To edit a category title, make the changes you want in the text field.

To edit the button text or button link URL, first you need to enable the fields but clicking on the checkbox - Enable to edit button fields.

To edit a button text, make the changes you want in the text field.

To edit a button link URL, make the changes you want in the text field.

If you do not want the URL link to open into a new tab, uncheck the checkbox next to Open Link in New Tab.

To edit the image URL, click on the upload image button. You can upload files to the media library. Or you can select an image from the media library. Click on the image and click on using this image button.

* Note that by default, WordPress Contributors cannot upload media (they don't have the upload_files capability).

If you no longer want the category to be seen on the site, make sure that the checkbox is unchecked for Enable this category to display it on the site.

Make sure to save changes.

Both back to list buttons and back to list link will take you back to the categories page.


3. Delete Category

To delete a category, you will find the Trash under the Category column within the table.

You can move multiple categories to the Trash at once.

Select the category you want to act on using the checkboxes, then select the action you want to take from the Bulk actions menu and click Apply.

The category will be moved into the trash. Once in the trash you can either restore the category or delete permanently.

If the Administrator sets the user up with delete permissions that said user will be allowed to delete the data.


4. Search Category

To search for a specific category or title.

Type out the wording that you want to search in the textbox next to the search categories button.

Next, click on the search categories button.

To see the full table click on the x in the textbox and click on the search categories button. 



Category Options:

1. Add Category Options

To create a new option, you will need to select a category from the dropdown list. Once you have chosen a category click on the select button.

By default, the icon is a pointing finger, however if you want to change that icon, use the show icon picker toggle

These said icons are the icons that are sat in front of the description for the facilities.

When adding a different icon to the category options, you can select from 60 different available icons.

The selected icon will show which icon is currently selected.

To unselect an icon, you will need to click on it again. The selected icon will show the default icon which will be the pointing finger.

To add a description to the category options, type the description into the field. e.g. Large clubhouse with cozy fireplace

Make sure that you click and add new option button.

Both back to list buttons and back to list link will take you back to the category options page.


2. Editing Category

To edit you will find the edit link under the category name in the table row. Click on edit to go to the edit category options section.

The category name cannot be edited. Once you have created the name it must be deleted either with the administrator, or the user must have delete permissions.

To change an icon to another icon, click on the desired icon. You have 60 icons to choose from.

To edit the description, type the desired wording that you want to have read on the front end. e.g. Large clubhouse with comfortable fireplace.

If you no longer want the category option to be seen on the site, make sure that the checkbox is unchecked for Enable this category option to display it on the site.

Make sure to save changes.

Both back to list buttons and back to list link will take you back to the category options page.


3. Delete Category

To delete a category within the category options table, you will find the Trash under the Category column within the table.

You can move multiple categories to the Trash at once.

Select the category you want to act on using the checkboxes, then select the action you want to take from the Bulk actions menu and click Apply.

The category will be moved into the trash. Once in the trash you can either restore the category or delete permanently.

If the Administrator sets the user up with delete permissions that said user will be allowed to delete the data


4. Search Category

To search for a specific category, icon or description.

Type out the wording that you want to search in the textbox next to the search category button.

Next, click on the search category button.

To see the full table click on the x in the textbox and click on the search category button. 




Permissions / Roles:

To add a role to create new data for the facilities click inside the text area and a dropdown will appear with the roles that you can select. Click on the role that you want to choose to add to the permission.

To remove a role, you click on the "x" next to the role. e.g. x|Editor or x|Author or x|Contributor.

You can take the same steps for edit the data and delete the data.

Make sure to save changes.


 
Tables:
The tables that the plugin creates are - facamen_categories, facamen_categoryoptions, and facamen_options.
